# 말투 룰

블로그 "쑤기로운 일상생활" 의 말투 패턴. 새 글을 쓸 때 이 톤을 유지한다.

## 종결어미

- **`~어요/예요/이에요`** 가 거의 모든 문장 종결 (존댓말 친근체)
- **`~았어요/었어요`** 과거형도 빈번 ("다녀왔어요", "방문했어요")
- **`~딱이에요` / `~딱!`** 가 시그니처 ("데이트 코스로 딱!", "회식 장소로 딱이에요")
- **`~돼요`** 안내문 끝 ("찾기 쉬워요", "이용 가능해요")
- **반말/명사형 종결 거의 없음** — 끝까지 존댓말 친근체

## 자주 쓰는 강조·감탄 표현

- `진짜` (강조 부사 1순위) — "진짜 맛있어요", "진짜 푸짐"
- `역시` ("역시 맛집", "역시 웨이팅 있더라고요")
- `딱` ("딱 좋아요", "들리기 딱인", "찾기 딱")
- `은근` ("은근 자주", "은근 푸짐")
- `꼭` ("꼭 먹어주세요", "꼭 한 번")
- `너무` ("너무 귀여워요", "너무 맛있어요")
- `~잖아요?` (공감 유도) — "~ 흔하지 않잖아요?", "~ 먹잖아요?"
- `~더라고요` (경험 회상) — "사람 많더라고요", "맛있더라고요"
- 의성어/의태어: 바삭바삭, 쫄깃쫄깃, 포슬포슬, 뜨끈뜨끈, 짭조름, 매콤, 고소, 담백

## 이모지 사용

- **드물게, 포인트로만** 사용 (한 글에 5~10개 정도)
- 위치/주차 안내 앞: `📍`, `🚗`, `🚘`
- 결제수단 앞: `💳`
- 강조 끝: `👍`, `~!!`
- 감성 묘사: `🌴` (남국), `❤️` 거의 안 씀
- 체크박스: `☑️` (대기 안내, 주문 팁)
- 부정: `❌️` ("닭 잡내 ❌️")

## 고정 마커 / 시그니처 표현

- **`#내돈내산`** — **협찬/체험단이 아닌 글에만** 본문 어딘가에 한 번 (가게정보 박스 직전 또는 직후)
  - ⚠️ **협찬·제공·체험단인 경우 절대 쓰지 말 것.** 표시광고법 위반 + 독자 신뢰 훼손
  - 협찬 글은 대신 **`소정의 원고료를 받고 작성된 글입니다`** 같은 명시 문구를 본문 상단 또는 하단에 넣음 (사용자가 별도 지정한 문구 우선)
- **`다녀왔어요`** — 인트로 마지막 문장
- **`참고하세요!`** / **`방문해 보세요`** — 마무리 문장
- **`~ 분들에게도 딱!`** — 추천 대상 명시할 때

## 문장·단락 길이

- **짧은 문장이 압도적.** 한 문장이 대부분 한 줄 (15~25자)
- 긴 내용도 짧게 끊어서 줄바꿈으로 분리
- 단락 사이에 **빈 줄(zero-width space) 한 줄** 또는 **두 줄**
- 예시:
  ```
  바삭한 식감에
  
  닭 잡내 없이
  
  깔끔한 마무리까지
  
  ​
  
  진짜 맛집 인정!
  ```

## 1인칭 / 시점

- **`저는` / `저`** 가끔 등장하지만 자주 생략
- "방문했어요" 식으로 주어 생략이 더 자연스러움
- "전 3인 방문이라" 같이 짧게 줄여 쓰는 경우도 있음

## 가격·정보 표기

- 영업시간: `매일 11:00-22:00` (한칸 공백, 24시간제, 하이픈)
- 가격: `5000원` / `최초 120분 2000원, 추가 10분당 1000원` 식
- 도보: `도보 5분 거리`
- 라스트오더: `라스트 오더 21:00`
- 정기휴무: `매주 화요일 정기휴무` 또는 `매주 화요일 휴무`

## 피해야 할 표현

- 격식체 (`~합니다`, `~입니다`) — 친근체와 안 어울림
- 과도한 미사여구 ("미식가의 영혼을 사로잡는...")
- 영어/외래어 남발
- 부정적·비난조 ("진짜 별로", "비추") — 솔직 후기여도 부드럽게

## 평가 시 톤

- **솔직하지만 부드럽게.** 단점도 적지만 "호불호 갈릴 수도" 정도로 완곡
- 칭찬 표현이 메인:
  - 맛: "진짜 맛있어요", "고소한 풍미", "균형 좋아요", "끝까지 질리지 않아요"
  - 양: "푸짐해요", "양 넉넉", "부족함 없어요"
  - 분위기: "분위기 좋아요", "테이블 간격 넓어서", "조용하고 깔끔"
  - 가성비: "가성비 갑", "가성비 좋아요"
