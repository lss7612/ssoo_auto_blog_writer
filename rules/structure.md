# 글 구조 룰

블로그 "쑤기로운 일상생활" 의 맛집 글에서 일관되게 반복되는 글 구성 순서. 새 글을 쓸 때 이 순서를 따른다.

## 전체 흐름

```
1. 표지 사진 (외관 / 메인 음식 1장)
2. 큰 제목 (H2 = 가게 이름 + 핵심 키워드)
3. 인트로 (1~3문장, 방문 동기/계기)
4. 가게 기본정보 박스
5. 위치 (+ 주차 별도 블록)
6. 매장 분위기
7. 메뉴 (+ 가격, 메뉴판 사진)
8. 음식 사진별 상세 후기 (메뉴마다 한 섹션)
9. 결제수단 / 화장실 등 잡정보 (선택)
10. 마무리 (총평 + 추천 대상)
11. 해시태그
```

## 각 섹션 디테일

### 1. 표지 사진
- 외관 또는 시그니처 메뉴 1장
- 사진 자리 placeholder: `<!-- PHOTO: 외관 또는 메인 음식 -->`

### 2. 큰 제목 (글 첫 줄)
- 형식: `{지역} {음식종류} {가게이름} | {핵심 키워드 2~3개}`
- 예: `강릉 사천해변 햄버거 모아나버거 | 하와이 감성 육즙 가득 수제버거`
- 페이지 가운데 정렬, 본문 첫 줄로 다시 한 번 적음 (네이버 SEO 의식)
- 본문 중간 섹션 위에도 같은 문장이 반복됨 (위치/주차/매장분위기/메뉴 직전 등)

### 3. 인트로 (1~3문장)
- "다녀왔어요" / "들렀어요" / "재방문 했어요" 로 끝나는 1문장이 핵심
- 방문 동기 (소문 듣고 / 박람회 갔다가 / 주민이라 / 데이트하다)
- `#내돈내산` 마커가 자주 등장 (특정 위치 없이 본문 어딘가)

### 4. 가게 기본정보 박스
- 가게명 (단독 줄, bold-like)
- 주소
- 영업시간 (`매일 11:00-22:00` 형식)
- 라스트오더, 휴무일
- 결제수단 (제로페이, 서울페이 등)
- 부가정보 (주차, 발렛, 포장, 단체이용, 유아의자 등)

### 5. 위치
- 소제목: "위치" 또는 "위치 & 주차"
- 가까운 지하철역 + 출구 번호 + 도보 분
- 한 줄 코멘트 ("데이트 코스로 딱!", "찾기 쉬워요")

### 6. 주차 (별도 블록 권장)
- 이모지 `🚗` 또는 `🚘` 로 시작하는 경우 많음
- 주차 가능 여부 / 주차장 위치 / 요금 (최초 N분 N원, 추가 10분당 N원)
- 발렛 가능 여부 / 만차 시 대안 (공영주차장 등)

### 7. 매장 분위기
- 소제목: "매장 분위기"
- 사진 1~2장
- 매장 크기, 테이블 수/배치 (2인석/4인석/룸)
- 분위기 형용사 (조용, 클래식, 깔끔, 하와이 감성)
- 누가 오기 좋은지 (소개팅 / 회식 / 가족모임 / 데이트)

### 8. 메뉴
- 소제목: "메뉴"
- 메뉴판 사진 1~3장 연속
- 가격대 또는 세트 구성 설명 (런치 vs 디너, 2인 세트 등)
- "콜키지 프리" / "리필 가능" 같은 특수 정책은 별도 섹션

### 9. 음식 후기 (메뉴 1개당 1섹션)
- 소제목: 메뉴 이름 그대로 (예: `모아나버거`, `잣 크림 리조또`)
- 사진 1~3장
- 구성 설명 (들어간 재료 나열)
- 맛 표현 (육즙·바삭·고소·담백·매콤 등)
- 한 줄 평가 ("호불호 없이 맛있어요" / "원픽이었어요")

### 10. 마무리
- 1~2문장
- "찾는다면 ~ 방문해보세요" / "~ 추천해요" 형식
- 추천 대상 다시 강조 ("데이트 코스로", "회식 장소로")

### 11. 해시태그
- 한 줄에 8~15개 모아쓰기 (`#지역맛집 #가게이름 #지하철역맛집 ...`)
- 띄어쓰기 없는 합성어 형태 (`#합정맛집` `#망원데이트`)

## 사진 배치 규칙

분석한 글들의 사진 등장 위치 (평균 15~20장):
- 표지: 외관 또는 메인 음식
- 위치 섹션: 가게 외관 또는 간판 클로즈업
- 매장 섹션: 인테리어 1~3장
- 메뉴 섹션: 메뉴판
- 음식 후기 섹션: 메뉴별 클로즈업 + 단면/들어올리기 샷

**사진은 본문 사이사이에 단독 줄로 들어감.** 텍스트 → 사진 → 텍스트 패턴.
