# 발췌 예시 (참고용)

본인 블로그에서 따온 짧은 발췌. 새 글을 쓸 때 톤·표현 참조용.

## 좋은 인트로 예시

```
강릉 사천해변 해안도로
따라 드라이브하다 보면 눈에 띄는

하와이 감성 제대로 느껴지는
수제버거 맛집 #모아나버거 다녀왔어요!
```

```
삼성 코엑스에서
열린 불교 박람회 들렸다가

분위기 좋다고 소문난
삼성동 코엑스 파스타 맛집
"치폴라로쏘 삼성본점" 다녀왔어요
```

```
합정 주민이라면 다들 아는
가성비 보쌈 맛집 "각시보쌈" 다녀왔어요-~

평일은 물론, 주말에도

대기는 무조건 생각하고
방문해야되는 찐맛집!
```

```
요즘 닭강정 먹고 싶을 땐

큐스닭강정과 망원닭강정
번갈아서 가고 있어요
```

```
영업시간 11:00-18:00
라스트 오더 17:30
매주 화요일 정기휴무
반려동물 동반, 포장, 주차
```

```
치폴라로쏘 삼성본점
서울 강남구 삼성동 91-25 지하 1층
영업시간 매일 11:00-22:00
라스트 오더 21:00
제로페이, 서울 페이, 예약, 포장, 주차,
발렛파킹, 대기공간, 단체 이용, 유아의자
```

## 좋은 위치/주차 표현

```
봉은사역 3번 출구
도보 1분 거리,
삼성역 도보 5분 거리 위치

코엑스 데이트 코스로
삼성동 파스타 맛집 찾는다면 딱!
```

```
🚘 주차

건물 주차장 이용 가능
바로 옆 이브자리
주차장 이용하면 됩니다

주차 최초 120분 2000원
이후 10분당 1000원 / 최대 2만 원
발렛 가능 (2000원)
```

## 좋은 매장 분위기 표현

```
테이블 수도 많고
클래식한 분위기의 규모가 큰 레스토랑이에요

2인석, 4인석, 룸까지
다양하게 준비되어 있고

조용하고 클래식한 분위기에
테이블 간격도 넓어서

분위기 좋은 식사 자리를 가능하기에
삼성동 코엑스 소개팅 장소로 딱!
```

## 좋은 음식 후기 표현

```
패티, 로메인, 적양파, 토마토, 치즈 그리고
스페셜 소스가 듬뿍 들어간 시그니처 햄버거

비주얼부터
소스가 흘러넘칠 정도로
푸짐해서 먹기 전부터
기대감 장난 아니었어요.

한입 딱 베어 무는
순간 육즙이 진짜 팡 터지는데

패티가 두툼하면서도
촉촉해서 씹을수록
고소한 풍미가
진하게 느껴졌어요
```

## 좋은 마무리 예시

```
강릉 햄버거 맛집 찾는다면
사천해변 #모아나버거 방문해 보세요~
```

```
분위기 좋은 삼성동 코엑스 파스타 맛집 찾는다면
"치폴라로쏘 삼성본점" 방문해 보세요
```
